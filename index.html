HTML Tables

Tag Name Description
<table> Table defines a table
<thead> Table Head The set of rows defining the column headers in a table
<th> Head cell Defines a header cell in a table
<tbody> Table body Groups the body content in a table
<tr> Table Row The table row container
<td> Table Data The table data container
<tfoot> Table footer The set of rows defining the footer in a table.

Table Attributes

Attribute Syntax Description
Colspan <td colspan="number "> The colspan atrribute defines the number of columns a cell should span
Rowspan <td rowspan="number "> The rowspan atrribute defines the number of rows a cell should span

CSS box model

The CSS box model is a box that wraps around an HTML element and controls the design and layout.

Property Syntax Description
box-sizing div{
border: 1px solid blue;
box-sizing: border-box;
}		 This property lets us include the padding and border in an element's total width and height.
